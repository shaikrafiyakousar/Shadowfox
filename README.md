| File | Topic | What it demonstrates |
|---|---|---|
| [Task1_Beginner/1_variables.py](Task1_Beginner/1_variables.py) | Variables | Declaring variables, checking data types, and calculating simple interest |
| [Task1_Beginner/2_if_condition.py](Task1_Beginner/2_if_condition.py) | If Condition | BMI classification, city-to-country lookup, and comparing two cities |
| [Task1_Beginner/3_for_loop.py](Task1_Beginner/3_for_loop.py) | For Loop | Random dice simulation and a jumping-jacks workout loop |
